## MOCs

He prefixes _MOC's_ with topic specific names like `YT Kanban` or `YT playlists`. 

other prefixes:
- **;** => articles
- **$** => tweets
- **@** => people
- **'** => videos
- **~** => podcasts
- **&** => books

## Shortcuts for  ★★★★★

He uses the **Templater** Plugin to create shortcuts for those symbols.

`★★★★★` => cmd+5
`★★★★✩` => cmd+4
`★★★✩✩`  => cmd+3
`★★✩✩✩` => cmd+2
`★✩✩✩✩` => cmd+1
